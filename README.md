# 智能成绩分析系统 - 完整功能版

🎓 专业级成绩分析工具，支持Excel/CSV导入和完整的高级功能

## 🚀 快速启动

### Windows 用户
**一键启动**
- 双击 `start.bat` - 自动安装依赖并启动服务器

**诊断启动**  
- 如果遇到问题，运行 `test-startup.bat` 诊断

### Mac 用户  
1. 双击 `auto-install-mac.sh`
2. 双击 `start.command` 启动程序

## 🔧 完整功能

### 核心功能
- ✅ Excel/CSV 数据导入分析
- ✅ 智能图表生成（客户端+服务端）
- ✅ 学生成绩统计排名
- ✅ 临界生智能识别

- ✅ 联表分析功能

## 📄 PDF功能说明

**PDF功能状态：**
- 系统会自动尝试安装PDF功能  
- 如果PDF功能不可用，不影响核心分析功能
- 可以使用数据导出和图表截图功能

**如果PDF功能可用：**
- 生成带图表的专业报告
- 支持A4格式，完美打印

## 🛠️ 自动安装

安装程序会自动安装：
- Node.js（如果未安装）
- Python（如果未安装，用于高级功能）  
- 所有必需的依赖包
- PDF生成相关库
